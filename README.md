一、编译方式
1、安装mysys2
2、make

运行方式

二、打开方式
1、双击打开fc.exe
2、把test.nes 目标rom 拖放在窗口中

三、操作方式
w、S、A、D 分别为上、下、左、右
u 是选择键
j 是A键
k 是B 键
Enter 是 start 键盘

问题:
当前只支持Windows 下的mysys2 编译。
